\# Governance



This repository is governed as an incubation branch policy asset.



\## 1. Claims control



\- No security or compliance guarantees

\- No implied certification

\- Scope limited to declared policy rules



\## 2. Change control



\- Policy changes must be explicit

\- Backwards-incompatible changes must be documented

\- Deletion preferred over expansion



\## 3. Elevation readiness



Elevation into WFSL requires:



\- enforcement capability (manual or automated)

\- reuse across multiple repositories

\- deterministic interpretation

\- a tagged elevation baseline



\## 4. Licence discipline



\- Apache License 2.0 only
